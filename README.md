# 基于SSH框架实现的鲜花订购系统

## 项目简介

> 1. 本系统使用Struts2+Spring+Hibernate架构，数据库使用MySQL,连接池使用c3p0。<br>
> 2. 模仿花礼网进行前端设计与开发，实现网站导航、商品分类展示，商品详情、商品检索、购物车等功能。<br>
> 3. 使用EasyUI实现后台对商品分类、商品信息、用户信息、订单信息的管理，包括增删改查，文件上传等。
